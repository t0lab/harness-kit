# Django

When working on Django projects, consult the skills at `.agents/skills/`:

- `django-patterns/` — architecture, DRF, ORM, caching, signals, middleware
- `django-security/` — auth, CSRF, SQL injection, XSS, secure deployment
- `django-tdd/` — pytest-django, factory_boy, mocking, DRF testing
- `django-verification/` — migrations, linting, tests, security scans, pre-release checks

Match the skill to the task. Run verification before PRs.
